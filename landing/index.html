Tính năng

Tất cả những gì bạn cần để phân tích thị trường Việt Nam

Từ phân tích cổ phiếu đơn lẻ đến kinh tế vĩ mô, tất cả được hỗ trợ bởi dữ liệu theo thời gian thực.

Phân tích cổ phiếu

Phân tích sâu — định giá, sức khỏe tài chính, sàng lọc, tâm lý và phân tích ngành cho bất kỳ cổ phiếu Việt Nam nào.

Phân tích kỹ thuật

20+ chỉ báo: SMA, EMA, RSI, MACD, Bollinger Bands, ATR, OBV và nhiều hơn với tín hiệu tự động.

Nghiên cứu thị trường

Định giá VNINDEX/VN30, so sánh ngành, cổ phiếu tăng/giảm mạnh, dòng tiền ngoại và phân tích quỹ.

Kinh tế vĩ mô

Dữ liệu GDP, CPI, lãi suất, tỷ giá, FDI và cán cân thương mại của Việt Nam.

Tâm lý tin tức

Thu thập tin tức theo thời gian thực từ CafeF, VnExpress và các trang tài chính Việt Nam với phân loại cảm xúc.

Phân tích quỹ đầu tư

58+ quỹ mở — theo dõi NAV, danh mục top, phân bổ ngành và so sánh hiệu suất.

Nguồn dữ liệu

12+ nhà cung cấp dữ liệu thời gian thực

Cổ phiếu Việt Nam, chứng khoán quốc tế, tiền mã hóa, vĩ mô, quỹ mở, hàng hóa và tìm kiếm web.

VCI Stock

Nguồn mặc định — đầy đủ dữ liệu cổ phiếu VN, tài chính, thông tin công ty

KBS Stock

Nguồn dự phòng — cùng phạm vi dữ liệu VN như VCI

MAS Stock

Báo giá, intraday, tài chính, độ sâu thị trường

TVS Stock

Dữ liệu tổng quan công ty cho cổ phiếu VN

VDS Stock

Dữ liệu giao dịch intraday với tự động cookie

FMP Stock

Cổ phiếu toàn cầu — báo giá, công ty, tài chính (cần API key)

BINANCE Crypto

Tiền mã hóa — lịch sử, intraday, độ sâu (không cần API key)

VND Market

P/E, P/B thị trường, cổ phiếu tăng/giảm mạnh VNINDEX

MBK Macro

GDP, CPI, lãi suất, FDI, cán cân thương mại

FMARKET Fund

58+ quỹ mở với NAV, danh mục và phân bổ

SPL Commodity

Vàng, dầu, thép, khí đốt, nông sản

Perplexity Search

Tìm kiếm web qua Perplexity API (cần API key)

Python 3.10+
