Features

Everything you need for Vietnamese market analysis

From individual stock deep dives to macro economics, all powered by real-time data.

Stock Analysis

Deep dive — valuation, financial health, screening, sentiment, and sector analysis for any Vietnamese stock.

Technical Analysis

20+ indicators: SMA, EMA, RSI, MACD, Bollinger Bands, ATR, OBV, and more with automatic signals.

Market Research

VNINDEX/VN30 valuation, sector comparison, top movers, foreign flows, and fund analysis.

Macro Economics

GDP, CPI, interest rates, exchange rates, FDI, and trade balance data for Vietnam.

News Sentiment

Real-time crawling from CafeF, VnExpress, and Vietnamese financial sites with sentiment classification.

Fund Analysis

58+ mutual funds — NAV tracking, top holdings, industry allocation, and performance comparison.

Data Sources

12+ real-time data providers

Vietnamese stocks, global equities, crypto, macro, mutual funds, commodities, and web search.

VCI Stock

Default provider — full VN stock data, financials, company info

KBS Stock

Fallback provider — same VN coverage as VCI

MAS Stock

Quote, intraday, financials, price depth

TVS Stock

Company overview data for VN stocks

VDS Stock

Intraday trading data with auto-cookie

FMP Stock

Global stocks — quote, company, financials (API key)

BINANCE Crypto

Crypto — history, intraday, depth (no API key)

VND Market

Market P/E, P/B, top movers for VNINDEX

MBK Macro

GDP, CPI, interest rates, FDI, trade balance

FMARKET Fund

58+ mutual funds with NAV, holdings, allocation

SPL Commodity

Gold, oil, steel, gas, agricultural commodities

Perplexity Search

Web search via Perplexity API (API key)
